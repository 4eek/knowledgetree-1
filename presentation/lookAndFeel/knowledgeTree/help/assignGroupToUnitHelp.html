‘Assign Group to Unit’

 

  1. Select the group to assign to a unit. Note that a group can only belong to a single unit.
  2. Select the unit to which the group belongs.
  3. Click the Select button.
    1. If the group already belongs to a particular unit, an error message will be displayed.
    2. Otherwise you will be prompted to confirm the assignment of the group. A message will indicate successful completion of this task
